# Pagaiacronos 

A toolset for retrieving, storing, and visualizing **Italian speed canoe race results** from the official FICR (Federazione Italiana dei Cronometristi) website.

---

## Repository Structure

```
<root>/
├── scraper/          # Python scraper module
└── app/              # Android application source (Pagaiacronos)



---

## Components

### 🐍 FICR Scraper (`scraper/`)

A Python module that retrieves speed canoe race results from the official FICR website and exports them to JSON.

**Features:**
- Fetches race results directly from the FICR website
- Exports structured data to `.json` format for downstream consumption
- Designed to create incremental builds of the database to keep data up to date

**Usage:**

```bash
cd scraper
pip install -r requirements.txt
python scraper.py --season <year> --out-file <output file> --incremental <incremental input file>
```


---

### 📱 Pagaiacronos Android App (`app/`)

An Android application that allows users to query and visualize canoe race results on their device.

**Features:**
- Browse and search race results by event, athlete, or category
- Local SQLite database for offline access
- Database sync from an external JSON file (e.g. downloaded from this repository's `data/` folder)
- Clean and intuitive UI tailored for canoe sports enthusiasts

**Building:**

Open the `app/` directory in Android Studio and build the project using Gradle

---

### 🗄️ Data 

The data generated by the scraper is stored in a JSON file. Due to the size of the file, currently the file is uploaded associated with the release of the application (data.json).  The app by default uses the URL of the release to retrieve the database associated with that release, but this can be changed through the settings

**Syncing from the app:**

1. Open Pagaiacronos on your Android device
3. Navigate to **Sync Database** (the URL of the file may be changed in the app settings)
4. Press sync (optionally you may clear the local copy)
